why do we need HTML?

HTML is essentially a markup language used to create documents that web browser can open and display as websites.

these documents are structured using HTML elements (tags), which define different parts of the content, such as headings, paragraph, images, links,tables and forms.

A web browser interprets the HTML code and displays it as a structured webpage. HTML documents are plain text files with the .html and .htmextension. HTML allows embedding of CSS and Javascript to enhance the webpage.
